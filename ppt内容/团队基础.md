## 1. 项目经验 · 硬件与算法基础 —— 小型自主遥控水下机器人 (ROV)

【PPT要点】
- 自主研发小型自主遥控水下机器人 (ROV)。
- 实现:
  - 推力分配 + 姿态自稳算法 → 定姿悬停 + 手动遥控。
  - 水下视觉增强算法 → 应对光线折射与畸变, 在非结构化环境中提取视觉特征。
  - 自建 Python 仿真平台 → 快速验证控制参数, 提升调试效率。

> 🖼 图片: ROV 实物照 (团队基础_ROV_1.png、团队基础_ROV_2.jpeg)

【讲稿】
在具身硬件与底层控制方面,我们自主研发过一款小型自主遥控水下机器人。控制上,我们融合推力分配与姿态自稳算法,实现了定姿悬停和手动遥控,积累了抗扰动控制经验;感知上,针对水下光线折射与畸变,我们做了专门的视觉增强算法,在非结构化环境中提取有效特征;验证上,我们自建了 Python 仿真平台,用于快速迭代控制参数。

## 2. 项目经验 · 软件基础 —— 教务智能体

【PPT要点】
- 教务智能体。
- 实现:
  - 动态更新的教务专属知识库。
  - 自然语言理解 + 按角色适配查询路径 + 语义拆解, 将模糊提问转为结构化指令。
  - 多智能体协作: 以「超级教务 Agent」为总调度, 协同数据查询、流程审批等子 Agent。
- 与本赛题的对应 (推论): 语义拆解为结构化指令、多智能体调度, 对应认知层大模型的任务卡解析与规划。

【讲稿】
在软件与大模型智能体方面,我们做过一个教务导索 Agent。它基于动态更新的教务知识库,能理解师生的自然语言提问,按不同角色适配查询路径,并通过语义拆解把模糊提问转成结构化指令。我们还实践了多智能体协作,用一个总调度 Agent 协同数据查询、流程审批等子 Agent。这套语义拆解与多智能体调度的经验,与本赛题认知层解析任务卡、输出结构化装配指令是同一类问题。
