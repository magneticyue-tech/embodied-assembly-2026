## 赛题(来源:materials/official/赛题.txt,与赛题.pdf 等同;附件为 赛题附件-数据流程图.png)

- 2026 中国大学生机械工程创新创意大赛 · 具身智能精密装配赛。当前阶段:校赛已完成方案论证,进入系统实现。
- 两个任务,无先后,都要执行,评审 5 分钟:
  - 任务一:任务卡1 场景识别 → 视觉识别 + 语音播报。
  - 任务二:任务卡2 装配指令识别 → 解析 6 方块色↔托盘色↔顺序 → 全自动装配。
- 强制项:唤醒词「小具同学」;应答「我已就绪,请下达指令」;解析过程存带时间戳文本;识别失败播报「任务卡识别失败,请确认任务卡位置」;单任务完成播报「任务已完成」;全部完成播报「两个任务均已完成」。
- 现场不提供网络;文本输入仅为备用,须申请且扣分;野蛮装配否决;删机器人系统文件判 0 分。
- 评审时间内任务失败或中断(含机器人故障、识别失败)时,需重抽任务卡1、任务卡2并重新开始,取最优成绩;不得在原任务上继续自动执行。
- 评分细则(materials/official/MarkingCriteria.txt):实时视觉界面、大模型带时间戳日志实时滚动、装配每步可观测。

## 硬件(来源:materials/aubo/ 各 PDF)

- 机械臂 AUBO-i5:6 自由度,负载 5kg,工作半径 886.5mm,重复定位精度 ±0.02mm(厂商标称)。随仓库提供的厂商手册列出 C/C++/Lua/Python SDK,未列出 Java SDK。
- 末端电动吸盘 + 挂载工业单目相机(eye-in-hand)+ 光源 + AI 边缘计算平台。
- 方块:6 个 30mm 塑料正方体(红橙黄绿蓝紫)。
- 托盘(实物照片 materials/official/troy.png):同心环精度尺,每环 1mm、每 5mm 加粗、量程 ±20mm;整块刚性板,2列×3行 6 色槽。
- 机器人仿真:官方 ARCS/AUBOPE(VMware + Python 脚本)。
