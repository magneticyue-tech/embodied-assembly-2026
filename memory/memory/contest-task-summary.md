校赛参赛。需从五方面阐述:赛题解析、思路介绍、成员介绍、团队基础、进度安排。最终用 Claude Design 出 PPT。项目根: E:\ClaudeCode\精密装配\ (环境 banner 误报 darwin/Users, 实为 Windows Git-Bash)。任务书 PDF: E:\ClaudeCode\精密装配\赛题.pdf

**赛题本质:** 具身智能 = 多模态感知(单目视觉+语音) → 大模型认知(理解+规划) → 机器人精密执行 → 反馈。核心是把大模型语义智能与机器人物理精度正确缝合。

**两个任务(无先后,都要做,5分钟评审):**
- 任务一:任务卡1场景识别 → 视觉识别+语音播报
- 任务二:任务卡2装配指令识别 → 解析6方块色↔托盘色↔顺序 → 全自动装配

**硬件(已确认):** 机械臂=**AUBO-i5**(6DOF,负载5kg,半径886.5mm,重复定位±0.02mm,SDK支持Python/C/C++/Java/Lua)+末端电动吸盘+挂载工业单目相机、光源、AI边缘计算平台、托盘、6个30mm正方体方块(红橙黄绿蓝紫)。机器人仿真=官方**ARCS/AUBOPE**(VMware+Python脚本)。可选视觉平台=海康**Vision Master**(无代码),本方案选自写OpenCV。资料在 精密装配资料/ 文件夹。

**托盘真实形态(troy.png):** 同心环精度尺——中心彩色方块+同心灰环,每环1mm、每5mm加粗、量程±20mm。放置偏移按"落在第几环"读数评分。方块/托盘可能旋转(边不与画面平行)→需minAreaRect旋转鲁棒检测。任务卡2示例(missionCardExample.png)。

机器人型号→AUBO-i5;②装配指令来源→相机读卡(VLM解析),ASR(唤醒/触发/复述)。

**评分细则(MarkingCriteria.txt):** 强制可观测三项——实时视觉界面(识别框+坐标提取)、大模型带时间戳日志实时滚动、装配每步可观测(语音播报/界面高亮);

**关键技术难点(用于赛题解析):**
1. 单目相机无深度 → 靠已知平面+已知方块尺寸+手眼标定做平面反投影定位(精密的命门)
2. 裁判在机器人前往拍照点期间可再次挪动方块/托盘 → 必须每块拍照定位的闭环再感知
3. 现场不提供网络 → 大模型/语音需离线或自备网络
4. 红橙、蓝紫色相接近 → 受控光源下 HSV/Lab 鲁棒分割
5. 全自动不可干预,失败重抽取最优 → 考鲁棒性与异常恢复
6. 强制项:唤醒词"小具同学"、应答"我已就绪请下达指令"、解析过程存带时间戳文本、识别失败语音报错

语义任务交大模型,度量任务交经典CV+几何。

**已产出的答辩材料(E:\ClaudeCode\精密装配\, 注意部分 md 已移入 ppt内容/ 与 方案与实现/):** 赛题解析.md、思路介绍.md(均为双栏:PPT要点+口语讲稿,供后续Claude Design做PPT用)、正向设计方案.md(已同步真工具链)、进度安排.md(四阶段:方案/认知+执行仿真/全链路联调+交互/真机部署)。成员介绍/团队基础由用户自行撰写。仿真见 [[sim-prototype]]。
