## 一、项目是什么

2026 中国大学生机械工程创新创意大赛 · **具身智能精密装配赛**(当前阶段:校赛 PPT 答辩)。
本质:把**大模型的语义智能**与**机械臂的物理精度**缝合成一条完整闭环,分四层:

**感知 → 认知 → 执行 → 交互**

- **任务一 · 场景识别**:相机拍工作区 → 大模型识别任务卡上的常见生活物品 → 语音播报。
- **任务二 · 依据指令装配**:大模型解析任务卡上 6 个「方块色→托盘色」映射与顺序 → 机械臂按序装配 → 语音播报。

## 二、关键事实(容易搞错,先记住)

- **精度瓶颈在视觉,不在机械臂**:AUBO-i5 重复定位 ±0.02mm,远高于任务需求;最终装配精度几乎全由**视觉定位**决定。
- **6 种固定颜色**:红、橙、黄、绿、蓝、紫。
- **托盘是一整块刚性板**,板上 2列×3行 共 6 个同心环色槽,共享一块板的位姿。视觉策略:
  - **颜色→槽位映射**:开局无遮挡时**一次性标定**,之后固定不变。
  - **板位姿**:每步靠**板外轮廓矩形**重新检测(整块板不会被小方块遮挡;裁判中途挪板也能跟上)。
  - 目标槽位置 = 板位姿 ⊕ 标定布局,**不靠重新识别颜色** → 方块盖住槽色/与他槽同色都不混淆。
- **同心环精度尺**:每环 1mm、每 5mm 加粗、量程 ±20mm;放置误差按"落在第几环"读出。
- **三大难点**:①物品中途被挪动 → 每步重新拍照定位;②方块盖住托盘颜色 → 靠板位姿而非颜色;③全程无人工干预 → 强鲁棒性。

## 三、目录结构

```
sim/                  纯软件仿真(无需真机/GPU),复现完整闭环,产出标注图+时间戳日志
  config.py           坐标系约定 + 硬件参数(AUBO-i5、同心环托盘)
  scene.py camera.py  仿真台面与相机采集
  vision.py           视觉:BGR→HSV 分割 → minAreaRect → 单应反投影到台面 mm(旋转鲁棒)
  cognition.py        认知层(大模型智能体);当前为规则桩,阶段二将接入真实 VLM
  execution.py        执行层:AUBO-i5 抓放 + 同心环误差读数
  interaction.py interfaces.py  交互层骨架 / 层间接口
  main.py             双任务状态机 + 逐步可观测
  output/             产物:标注图 + parse_log_*.txt(赛题强制要求的时间戳日志)
ppt内容/              答辩五部分文稿:赛题解析 思路介绍 成员介绍 团队基础 进度安排
方案与实现/           正向设计方案 + 数据流程图
memory/               Claude 记忆(团队协作共享,勿忽略)
精密装配资料/         赛方原始 PDF(AUBO-i5、ARCS/AUBOPE、Vision Master 等)
troy.png missionCardExample.png  同心环精度尺 / 任务卡样例
MarkingCriteria.txt 赛题.pdf 校赛内容.txt  赛题与评分原文
```

## 四、运行仿真

```bash
cd sim && python main.py
```
依赖:`numpy opencv-python pillow`(Python 3.14 可装)。缺包:`python -m pip install numpy opencv-python pillow`。

## 六、仿真边界(答辩要诚实说明)

- **是**:验证视觉/认知/调度逻辑正确性,端到端跑通,产出可视化与日志。
- **不是**:物理动力学仿真(吸附力/碰撞/关节动力学未建模;pybullet 在 Py3.14 无预编译包,真机用 AUBO SDK + ARCS)。
- 本仿真单应理想,误差 <0.5mm;真实赛场视觉误差更大,同心环精度尺正是用来读这个误差的。
